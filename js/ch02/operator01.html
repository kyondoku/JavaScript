연산자(operator)

1. 산술연산자(+, -, *, /, %)
- / : 나누어서 몫, % : 나누어서 나머지
2. 증감연산자 (++, --)
- 증가연산자(++) : 변수의 앞 또는 뒤에 ++를 붙여 사용, 전위증가연산자, 후위증가연산자 - 감소연산자(--) : 변수의 앞 또는 뒤에 --를 붙여 사용. 3. 비교(관계)연산자 : >, >=, <, <=, ==, !=, ===, !==
- 결과가 논리값 형태로 표현, true 또는 false
- ===, !== : (자바스크립트에서 추가) 값과 타입을 한꺼번에 비교하여 true, false
