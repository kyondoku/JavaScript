연산자(operator)

3. 비교연산자 : >, >=, <, <=, ==(equal), !=, ===, !==
- 우선순위 : (>, >=, <, <=) 연산자가 (==, !=)보다 우선임.
- 우선순위 : (==, !=) 연산자가 (===, !==)보다 우선임.
- (>, >=, <, <=) > (==, !=) > (===, !==)
4. 논리연산자 : &&(논리 AND), ||(논리 OR, 논리합), !(논리 NOT, 논리 부정) - 논리연산자의 결과는 논리값(true, false 둘 중에 하나)
- 비교연산자가 논리연산자보다 우선임. 비교 > 논리
- 논리 AND가 논리 OR보다 우선임.
5. 대입연산자 : =
복합대입연산자(할당연산자, assignment) : +=, -=, *=, /=, %=
